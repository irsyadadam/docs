## Model Families

<CardGroup cols={2}>

<Card title="SMB-v1-Structure" icon="star" href="/models/standard-model-v1">

**1.7B parameters** · Biological World Model

Our flagship multimodal model that predicts patient trajectories using JEPA architecture. Integrates genomics, imaging, EHR, and proteomics.

```
standardmodelbio/SMB-v1-1.7B-Structure
```

</Card>

<Card title="SMB-EHR" icon="file-medical" href="/models/smb-ehr">

**4B parameters** · Electronic Health Records

Foundation model for clinical event prediction and temporal reasoning over patient records.

```
standardmodelbio/smb-ehr-4b
```

</Card>

<Card title="SMB-Vision" icon="eye" href="/models/smb-vision">

**97M - 600M parameters** · Medical Imaging

Vision encoders for radiology, pathology, and CT imaging tasks. Multiple sizes available.

```
standardmodelbio/smb-vision-base
```

</Card>

<Card title="SMB-Language" icon="message" href="/models/smb-language">

**8B parameters** · Clinical Text

Biomedical language model for sentence similarity, semantic search, and text understanding.

```
standardmodelbio/model-model-smb-mntp-llama-3.1-8b-v1
```

</Card>

</CardGroup>

## Choose Your Model

<AccordionGroup>

<Accordion title="I want to predict patient outcomes over time" icon="chart-line">

Use **Standard Model v1** — our JEPA-based world model that predicts how patients evolve given interventions.

[View Standard Model v1 documentation →](/get-started/models#standard-model-v1)

</Accordion>

<Accordion title="I'm working with electronic health records" icon="file-medical">

Use **SMB-EHR-4B** — trained on longitudinal clinical events for next-event prediction and temporal reasoning.

[View SMB-EHR documentation →](/get-started/models#smb-ehr)

</Accordion>

<Accordion title="I need to analyze medical images" icon="image">

Start with **SMB-Vision** models. Start with `smb-vision-base` for general tasks, or `smb-vision-ct-base-0519` for CT scans.

[View SMB-Vision documentation →](/get-started/models#smb-vision)

</Accordion>

<Accordion title="I'm doing clinical text analysis" icon="file-lines">

Use **SMB-Language-8B** for semantic search, sentence similarity, and clinical text embeddings.

[View SMB-Language documentation →](/get-started/models#smb-language)

</Accordion>

</AccordionGroup>

## Hardware Requirements

| Model Size | GPU Memory | Examples |
|------------|------------|----------|
| ~100M | 4-8 GB | smb-vision-base, smb-vision-ct-base |
| ~300M-600M | 8-16 GB | smb-vision-large, smb-vision-v0-risk |
| 1.7B | 16-32 GB | Standard Model v1 |
| 4B | 24-48 GB | smb-ehr-4b |
| 8B | 32-80 GB | smb-language |

<Tip>

See the [Quickstart guide](/get-started/quickstart#reducing-memory-usage) for tips on reducing memory with float16 and quantization.

</Tip>
